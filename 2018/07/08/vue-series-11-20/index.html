Intro to Components

A component is an reusable piece of code. When we’re creating a component there’s going to be multiple instances using the same kind of definition and if we just use an object instead of a function here, then all the instance will share the object by reference .If we change one of the instance then it’s going to update in every single component. When using function, we just calling this function to return a fresh copy of data.
