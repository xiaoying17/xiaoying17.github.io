浮动

元素设置为浮动后，其在文档流中占有的位置会被删除。设置浮动后的元素会形成BFC，使得内部的元素不会被外部所干扰，并且元素的宽度不再自适应父元素宽度，而是适应自身内容。
当容器的高度为auto，容器内容中有浮动元素时，容器的高度不能自动伸长以适应内容的高度，使得内容溢出到容器外边而影响布局。称为浮动溢出。子元素浮动之后会引起父元素高度的塌陷。清除浮动常用方法:对父元素的overflow设置为hidden;对子元素添加一个空标签，并设置clear:both;对子元素的after伪类进行设置(这个还要学习一下);对父元素也设置float。

双飞翼布局

圣杯布局

二者在html中均按照middle-left-right的顺序。
在css中，先分别将三者设为浮动；middle块宽度自适应width：100%；left块设置margin:-100%;right块设置margin:-宽度。
双飞翼中，在middle块内部再添加一个标签middle-inner，并设置其margin-left margin-right使其内容显示宽度正好位于中间可视范围。最后，可对父元素添加overflow hidden清除浮动。此时容器也就有了高度。
圣杯布局中，对父元素容器设置padding来使三者整体缩进，进而使用相对定位让left和right部分回到缩进留出来的位置。
