今日份的学习--八大排序

发表于 2018-01-22

冒泡排序

第一轮：从arr[0]开始，两个元素依次相互比较，将最大值冒泡到最后一位;第二轮相互比较的元素从arr[0]至arr[len-1],这一轮的最大值最后到了arr[len-1]处，经过len-1轮排序完成。

快速排序

可以选取数组的中间元素为pivot，将数组中小于pivot的元素放在pivot左边的left数组中，大于pivot的元素放在pivot右边的right数组中，然后依次递归。

以上两种为交换排序。

简单选择排序

每一轮都从操作数组的第一个元素开始为基准，选择出最小的元素，将这个元素与基准元素交换。
[7, 5, 3, 11, 2, 1] –> [1, 5, 3, 11, 2, 7] –> [1, 2, 3, 11, 5, 7] –> [1, 2, 3, 7, 5, 11] –>[1, 2, 3, 5, 7, 11]

堆排序

1:构建大顶堆(从最后一个非叶节点开始调整)
2:将堆的根节点与最后一个叶节点互换，在数组中的表现为最大值到了数组最后一位
3:重新调整堆，不考虑已经归位的元素。
注意，调整堆的过程也是递归的过程。比如在调整根结点后，其子节点如果不满足大顶堆的规则，则也要调整。

注意heapSort中的循环。
以上两种为选择排序。

直接插入排序

打牌的思路，每拿到一张牌与前面已经排好序的牌从右至左比较，插入到合适的位置。
[7, 5, 3, 11, 2, 1] –> [7][5, 3, 11, 2, 1] –> [5, 7][3, 11, 2, 1] –> [3, 5, 7][11, 2, 1]

希尔排序

优先比较距离较远的元素。核心在于间隔序列的设定。并没有针对每一轮gap后比较的结果进行合并，而是原始数组上，在每一个常数gap上，插入排序间隔为gap的元素。插入排序从1开始，希尔排序从gap开始，gap+1时相当于另一个插入排序，放在同一个循环中互不影响，因为间隔是一定的。

以上为插入排序。
