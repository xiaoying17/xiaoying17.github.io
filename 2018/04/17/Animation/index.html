Animation

发表于 2018-04-17

Animation定义: 1、Instill with life 2、Having the appearance of movement.
Css Movement包括 1、Transition Property 2、Animation Property + Keyframes
Elements that you should animate(UX)

Part 1 Transitions

Performant Properties: position scale rotation opacity,不会引起重流重绘。
Triggering: 1、Hover pseudo class 2、Class changes

Part 2 Keyframes

Keyframes is what animation does, and the animation is how it’s done,like speed ,time.

Triggering: 1、Hover pseduo class 2、Class changes 3、new elements
注意transform是个属性，可以scale放大缩小、rotate旋转等，动画是从一个状态到另一个状态。

When to Animate

1.Orientation orient people to where they are from where they were,用户从一个界面到另一个界面，两个界面用这个动画连接。
2.Functional Change answers what does this element do now。eg:button有on、off状态，可以清楚地显示现在是在哪个状态。
3.New Element help people know something new happened here.
4.Highlight little pulsing bring our attention to that spot
5.Visual Feedback eg:mac系统邮件点击发送后，提交界面向上滑动，有个模拟发送的效果。make the layout understand what I am asking it to do. animation represent that idea with motion.
6.System Status eg:进度条 let users know we aren’t blocked
